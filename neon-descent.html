FIREBASE SETUP

1
Go to console.firebase.google.com and create a new project
2
Click Build → Realtime Database → Create Database and choose a region. Select Test mode to allow read/write.
3
In Project Settings → Your Apps, click the web icon </>, register an app, and copy the config values.
4
Open this HTML file in a text editor. Find the line const FB_CONFIG = { near the top of the script and paste your apiKey, databaseURL, and projectId.
5
The databaseURL looks like https://yourapp-default-rtdb.firebaseio.com — it only appears after you create the Realtime Database.
6
Save and upload to GitHub Pages. Done — all players share the same global leaderboard automatically.
